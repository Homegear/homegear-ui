# Shif - Smarthome Interface

## Requirements
### Server
    php 7+

### Client
    ECMAScript 6+ 2015+
    WebSocket
    ViewPort

## Build environment setup:
    apt install php-cli npm
    npm -g install npx

## Build
    npm install
    npm run build

## Configure
### Settings
    to define individual settings create: /interfacedata.custom.php
